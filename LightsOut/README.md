# Lights Out

The classic 5×5 light-toggle puzzle, rebuilt with the HTML5 Canvas 2D API.
Press a light to flip it **and** its neighbours; switch every light off to win.

## How to play

Open `index.html` in any modern browser — no build step or server required.

- **Click / tap a tile** to toggle it and its four orthogonal neighbours.
- Turn **every** light off to solve the board and advance to the next, harder
  level.
- **N** deals a new board at the current level; **R** resets the current board
  to how it started.

Solve in as few moves as possible — your best (fewest moves) is saved in the
browser and shown in the header.

### Why every board is solvable

Boards are generated by starting from an all-off grid and applying random
presses. Because a press is its own inverse, any board produced this way can
always be cleared — you will never be handed an impossible puzzle.

## How it works

See [DESIGN.md](DESIGN.md) for the concept, mechanics and architecture. The game
rules are small pure functions over a global 5×5 boolean grid, kept separate
from rendering, which makes the game fully deterministic and easy to test.

## Development

Tests are written with Playwright and live in `tests/`. From the repository
root:

```bash
npm install
npx playwright install chromium   # first time only
npx playwright test LightsOut/tests/
```
