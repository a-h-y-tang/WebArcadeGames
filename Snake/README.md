# Snake

A classic Snake game built with HTML5 Canvas.

## How to Play

Open `index.html` in any modern browser — no build step, no dependencies.

| Input | Action |
|---|---|
| Arrow keys or WASD | Steer the snake |
| P | Pause / resume |
| Any arrow / WASD | Start or restart |

**Objective:** Eat the red food to grow longer and increase your score. Avoid hitting the walls or your own tail.

**Speed:** The snake gets faster every time you eat — plan ahead!

Your best score is saved in `localStorage` and persists between sessions.
